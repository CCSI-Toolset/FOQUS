CCSI Software used by FOQUS

Turbine Gateway

Turbine https://www.acceleratecarboncapture.org/drupal/product/turbscigate

TurbineLite is used to run process simulations for FOQUS on the local machine. More information can be found by following the links above.

A Turbine gateway can also be set up to run simulations remotley and in parallel.

SimSinter

https://www.acceleratecarboncapture.org/drupal/product/ccsisinter

SimSimter is used by the turbine gateway to run simulations. A SimSinter configuration file is required along with the process simulation to specify the inputs and outputs of interest. Although SimSinter is not required to use FOQUS, it comes with graphical tools to create sinter configurations files. The SimSinter documentation also provides details about how it works, and the format of the sinter configuration file. SimSinter may be downloaded from the link above.

PSUADE

PSUADE Page: http://computation.llnl.gov/casc/uncertainty_quantification/
PSUADE Windows Download: https://www.acceleratecarboncapture.org/drupal/product/psuadew

PSUADE is installed with the FOQUS bundel and provided under the Lesser Genral Public Licese (LGPL). See the PSUADE web page for more details.

ALAMO

https://www.acceleratecarboncapture.org/drupal/product/alamo-fortran http://archimedes.cheme.cmu.edu/?q=alamo http://minlp.com/
